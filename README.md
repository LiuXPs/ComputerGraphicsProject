# Draw3D

## 依赖项

- Qt6

- MinGW编译器

- Windows、Linux

## 功能

- `绘制`二维多边形，可以`指定坐标输入`、`平移`、`旋转`、`镜像`、`缩放`等二维编辑

- 将二维凸多边形`拉伸`成三维几何体，可以进行`旋转`、`平移`、`显示类型`、`渲染设置`、`路径动画`等三维编辑
